# Mobile (Expo + React Native)

Aplicativo principal do aluno/professor.

## Fluxos sugeridos para MVP

- Onboarding: idioma nativo, idioma alvo, meta (viagem, trabalho, prova de proficiência).
- Diagnóstico inicial (texto + áudio curto).
- Home com plano diário e streak.
- Aula conversacional com IA (chat + voz).
- Correção de tarefas e acompanhamento de evolução.
